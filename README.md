# Shikanox - Personal GNU/Linux

![Shikanox Screenshot](./assets/screenshot.png)

A complete personal config "distro" built on GNU Guix System, GNU Guix Home and Nix Home Manager.

### This repo defines:

- **Shikasystem** - GNU Guix System configuration
- **Shikanhome** - GNU Guix Home configuration
- **Shikapkgs** - GNU Guix Packages channel
- **Shikanix** - Nix Home Manager configuration
- **Shikadotfiles** - Dotfiles
