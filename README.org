A personal config for [[https://codeberg.org/ch4og/shikanox][Shikanox]] "distro" built on GNU Guix System, GNU Guix Home and Nix Home Manager.

* Dotfiles

Personal dotfiles are organized using a layout compatible with [[https://www.gnu.org/software/stow/manual/stow.html#Introduction][GNU Stow]].

Each subdirectory here represents a *"package"* of configuration files,
automatically linked into =~= via the =blue dots=.

** Usage

#+begin_src sh
  cd dotfiles
  stow fish mangowc other-package-name -t ~ --dotfiles
#+end_src

* Blueprint file

This repository uses [[https://codeberg.org/lapislazuli/blue][blue]].

Instead of running long guix and nix invocations manually, you can for example do:

#+BEGIN_SRC sh
  blue system
  blue style
  blue nix up
#+END_SRC

(Hacky blue wrapper binary loaded to =$PATH= with direnv)
